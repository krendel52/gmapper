# gmapper

A Go code generator that creates struct mapping functions between packages.

## What it does

Given two structs in different packages, `gmapper` generates a function that copies fields with matching names from one struct to the other. Fields that exist only in the source or only in the destination are silently skipped.

**Input:**

```go
// package domain
type DomainModel struct {
    ID    int64
    Name  string
    Count int
    Extra string // only in source — ignored
}

// package api
type ApiModel struct {
    ID     int64
    Name   string
    Count  int
    Extra2 string // only in destination — ignored
}
```

**Generated output** (`map_domainmodel_to_apimodel.go`):

```go
// Code generated by gmapper; DO NOT EDIT.

package api

import "github.com/krendel52/gmapper/example/domain"

func MapDomainModelToApiModel(src *domain.DomainModel) *ApiModel {
    return &ApiModel{
        ID:    src.ID,
        Name:  src.Name,
        Count: src.Count,
    }
}
```

## Installation

```sh
go install github.com/krendel52/gmapper@latest
```

## Usage

Add a `//go:generate` directive next to the destination struct:

```go
package api

//go:generate go run github.com/krendel52/gmapper -from github.com/your/module/domain.DomainModel -to ApiModel

type ApiModel struct {
    ID   int64
    Name string
}
```

Then run:

```sh
go generate ./...
```

The generated file is placed in the same directory as the file containing the directive.

## Flags

| Flag | Description | Example |
|------|-------------|---------|
| `-from` | Source type as `importpath.TypeName` | `github.com/user/pkg/domain.User` |
| `-to` | Destination type name in the current package | `UserDTO` |

## How field matching works

Fields are matched by **name**. Type compatibility is checked by the Go compiler when you build — if types differ, you'll get a compile error pointing to the exact field.

| Source field | Destination field | Mapped? |
|---|---|---|
| `ID int64` | `ID int64` | Yes |
| `Extra string` | *(absent)* | No |
| *(absent)* | `Extra2 string` | No |
